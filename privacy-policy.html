Política de Privacidade da Extensão "Minha Rua Segura"

1. Coleta de Dados

A extensão Minha Rua Segura não coleta dados pessoais dos usuários. Todos os dados exibidos são obtidos de fontes públicas disponibilizadas pela Secretaria de Segurança Pública do Estado de São Paulo (SSP-SP).

2. Uso dos Dados

Os dados são utilizados exclusivamente para fornecer informações sobre furtos em determinadas áreas. Não compartilhamos, vendemos ou utilizamos os dados para qualquer outro propósito.

3. Segurança dos Dados

Embora não coletemos dados pessoais, estamos comprometidos em garantir a segurança e a privacidade dos usuários. Recomendamos que os usuários sempre verifiquem as informações com fontes oficiais.

4. Alterações na Política de Privacidade

Esta política de privacidade pode ser atualizada periodicamente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados sobre como estamos protegendo suas informações.

Avisos Finais

  1. A extensão não modifica o site do Quinto Andar, apenas exibe dados complementares.
  2. A SSP-SP não endossa ou valida esta extensão - os dados são utilizados conforme disponibilizados publicamente.
  3. Para decisões críticas, sempre consulte registros oficiais e profissionais qualificados.
